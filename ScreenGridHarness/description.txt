A native OS automation harness for AI-driven screen control. Captures screenshots with coordinate grid overlays at multiple densities (coarse to ultra-fine), translates spreadsheet-style cell references (A1, B4, AA15) to pixel coordinates, and executes native mouse/keyboard input via PyAutoGUI. Supports multi-monitor capture, click/move/type/key/scroll actions, and maintains action history. Designed as the input/output bridge for AI agents that need to see and interact with any application.
