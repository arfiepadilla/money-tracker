Example Tool Server - Reference Implementation

This workflow demonstrates how to use the ContextUI tool registry system. It provides:

- A visual interface for starting/stopping the example Python tool server
- Real-time display of registered tools from the central registry
- An interactive tool tester for executing tools with custom arguments
- A live log view showing server activity

The example server registers 4 demo tools:
  - greet: Generate personalized greetings
  - calculate: Perform basic arithmetic operations
  - echo: Echo back messages
  - delay: Async delay example

Use this workflow as a reference when building your own tool-providing agents.
The Python server code is located in: example_modules/_shared/example_tool_server.py
