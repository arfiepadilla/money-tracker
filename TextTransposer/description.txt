Text Transposer

A simple utility to transpose text input by converting rows to columns and columns to rows.

Features:
- Transpose by Lines: Treats each line as a row, creates columns from characters
- Transpose by Words: Treats each word as a column, creates rows from characters
- Real-time preview of transposed output
- Copy to clipboard functionality
- Server status monitoring

Example (Lines Mode):
Input:
ABC
DEF
GHI

Output:
ADG
BEH
CFI

Example (Words Mode):
Input: hello world

Output:
hw
ee
ll
ll
oo
wr
dl
